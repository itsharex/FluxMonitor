Flux Remote Privacy Policy

This policy applies to Flux Remote for Android, the mobile client used to connect to a Flux Monitor server configured by the user.

Last updated: May 25, 2026

1. Overview

Flux Remote is a remote monitoring and management client. It connects to Flux Monitor servers that you add in the app. The developer does not operate a central account service for Flux Remote and does not collect personal data from the Android app for advertising, tracking, or analytics.

2. Data Stored on Your Device

Flux Remote may store the following information locally on your Android device so the app can connect to your server:

  • Server name, address, port, and connection settings.
  • Login credentials or tokens that you enter for your own Flux Monitor server.
  • App preferences such as language, theme, and display settings.

This data is used only to provide the app's remote connection features. It is not sold or shared by the developer.

3. Data Sent to Your Server

When you use Flux Remote, the app sends requests directly to the Flux Monitor server address you configure. These requests may include login credentials, commands, configuration changes, or other management actions that you initiate. The data is handled by your own Flux Monitor server and by the network path between your Android device and that server.

4. Third-party Services

Flux Remote for Android does not include third-party advertising or tracking SDKs. If you connect to a server through a public tunnel, reverse proxy, VPN, DNS provider, or other network service, that service may process network traffic according to its own policy. If your Flux Monitor server uses optional AI features, those requests are processed by the AI provider configured on the server side, not by Flux Remote itself.

5. Permissions

Flux Remote needs network access to connect to your configured server. The app does not request access to contacts, precise location, microphone, camera, or photos for its core monitoring features.

6. Security

You are responsible for the security of the Flux Monitor server you connect to. We recommend using HTTPS or a trusted private network, setting a strong password, keeping the server app up to date, and removing servers you no longer use from the app.

7. Data Deletion

You can delete saved server profiles and app data from within Flux Remote if the app provides those controls, or by using Android system settings to clear the app's storage or uninstall the app. Data stored on your Flux Monitor server must be deleted on that server.

8. Children's Privacy

Flux Remote is not directed to children. The app is intended for users who manage their own computer or server environment.

9. Changes to This Policy

We may update this policy when app behavior changes or when legal requirements change. The updated version will be posted on this page with a new last updated date.

10. Contact

If you have questions about this policy, contact the developer through GitHub:
