Mac server control plane Mac 服务器控制面板

Monitor, manage, and reach your Mac server. 监控、管理、远程连接你的 Mac 服务器。

Flux pairs a native macOS server launcher with a web dashboard and Flux Remote for iPhone. Watch system health, manage ports, inspect logs, control Docker and Nginx, and keep the machine reachable.

浮光由 macOS 浮光启动器、Web 控制面板和 iPhone 浮光远控组成。你可以查看系统状态、管理端口、分析日志、控制 Docker 与 Nginx，并让 Mac 保持可访问。

One stack, two clear roles.

一套系统，两个端清楚分工。

Run the server side on your Mac. Use Flux Remote when you need status and operations away from the desk.

浮光启动器运行在 Mac 上，负责启动和管理服务；浮光远控安装在 iPhone 上，用于远程查看和操作。

01 Server

Start the macOS launcher

启动浮光启动器

The native launcher runs the monitor service, keeps a menu bar entry, and opens the local dashboard.

原生启动器负责运行监控服务，提供菜单栏入口，并打开本地控制面板。

02 Dashboard

Operate from the browser

在浏览器内管理

Inspect CPU, memory, processes, ports, logs, configs, LaunchAgents, Docker, and Nginx in one place.

在一个界面里查看 CPU、内存、进程、端口、日志、配置、LaunchAgent、Docker 和 Nginx。

03 Remote

Control from iPhone

用 iPhone 遥控

Flux Remote connects to your server so you can check status and handle common operations on the go.

浮光远控连接浮光启动器，让你在手机上查看状态并处理常用操作。

Built for Macs that run services.

为承担服务的 Mac 设计。

Operational views without the admin clutter.

少一点杂乱，多一点可操作信息。

The interface keeps monitoring, configuration, and action flows close together, so checking the machine and fixing common issues stay in the same context.

界面把监控、配置和操作放在同一个工作流中，查看机器状态和处理常见问题不需要频繁切换工具。

  • Public access through InstaTunnel when you need an external route. 需要外部访问时，可通过 InstaTunnel 开启公网入口。
  • Optional OpenAI configuration for logs, config review, and troubleshooting. 可选 OpenAI 配置，用于日志、配置审查和故障分析。
  • Homebrew cask and DMG options for the macOS server side. 浮光启动器同时提供 DMG 和 Homebrew Cask 安装方式。

Runs on the Mac server. Starts the local monitor service, keeps the dashboard available, and exposes management modules for the machine.

运行在 Mac 服务器上。负责启动本地监控服务、保持控制面板可用，并提供这台机器的管理模块。

Remote-side

浮光远控 iOS 端

运行在 iPhone 或 iPad 上。连接浮光启动器后，可以离开电脑查看状态并执行常用操作。
